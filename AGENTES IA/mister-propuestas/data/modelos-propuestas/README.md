# Modelos de Propuestas Aprendidos

Este directorio contiene modelos reales de propuestas presentadas en licitaciones SEACE
que sirven como referencia para el agente de IA.

## Archivos

1. **consorcio-tec-piscobamba.json** - Propuesta de consorcio para consultoría de expediente técnico

## Uso

Estos modelos se utilizan para:
- Entrenar el RAG con ejemplos reales de formato
- Validar la estructura de anexos generados
- Aprender patrones de redacción legal
